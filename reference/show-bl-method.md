# Basic show method for bl class

Displays the contents of a Bayes linear ('bl') object. Called
automatically when an object of class 'bl' is shown in the console.

## Usage

``` r
# S4 method for class 'bl'
show(object)
```

## Arguments

- object:

  The object of class 'bl' to show.

## Value

Invisibly returns the original object `object`.
